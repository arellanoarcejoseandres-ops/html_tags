Guía Práctica de Etiquetas HTML

En esta página explicaremos el uso de etiquetas HTML básicas y avanzadas con ejemplos prácticos.

Etiquetas Básicas

<html>

Define el documento HTML y contiene todas las demás etiquetas. Es el contenedor raíz.

Ejemplo: <html lang="es"> ... </html>

<head>

Contiene metadatos, como el título y enlaces a estilos. No se muestra en la página.

Ejemplo: <head><title>Título</title></head>

<body>

Contiene el contenido visible de la página, como textos, imágenes y enlaces.

Ejemplo: <body><p>Hola</p></body>

<h1>

Define un encabezado de nivel 1 (el más importante).

Ejemplo:

Este es un encabezado

<p>

Representa un párrafo de texto.

Ejemplo:

Este es un párrafo de ejemplo.

<a>

Crea un enlace hipertexto a otra página o recurso.

Ejemplo:

Visita W3Schools

<img>

Inserta una imagen en la página. Requiere el atributo src.

Ejemplo:

<ul>

Crea una lista desordenada (con viñetas).

Ejemplo:

  • Elemento 1
  • Elemento 2

<ol>

Crea una lista ordenada (numerada).

Ejemplo:

  1. Paso 1
  2. Paso 2

<table>

Define una tabla para mostrar datos estructurados.

Ejemplo:

Encabezado 1 Encabezado 2
Dato 1 Dato 2
